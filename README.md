Project Overview
This is a simple Employee Management System built with Node.js, Express, MongoDB for the backend and Angular for the frontend.  
Users can register, login, and manage employees (add, update, delete, view).

---

Installation:
  Frontend :  
          Create an Frontend folder and change to that folder using cd Frontend
          create angular app using ng "new employee-managemaent"
          then cd employee-management
          create child components using "ng generate component registe.component"
          "ng serve" to run an code

  Backend:
          Create an Backend folder and change to that folder using cd Backend
          start with "npm init"
          create backend folder 
          Then install packages using "npm install express"
          Create an main folder named as "Server.js"
          Run the code using "node Server.js"
  
Features
- User Registration & Login  
- JWT Authentication & Authorization  
- Add, Edit, Delete, and View Employee Records  
- Role-based access (optional)  
- Responsive UI using Angular standalone components  

---

Tech Stack
Backend:
- Node.js  
- Express.js  
- MongoDB / Mongoose  
- JSON Web Tokens (JWT)  

Frontend: 
- Angular   
- Angular Router & FormsModule  
- TypeScript & HTML/CSS
